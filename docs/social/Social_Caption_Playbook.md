# Social Caption Playbook (Instagram + Facebook + TikTok)

Reusable caption system for **Next Gen Pickleball Academy** on Instagram
(`@nextgenpickleballacademy`), Facebook, and TikTok. Goal: post-ready, on-brand
captions in under five minutes that read like a human coach wrote them — never
like AI.

**Source of truth:** `BRAND_GUIDELINES.md` (COPYWRITING GUARDRAILS) and
`src/data/site.ts` (handles, tagline, boilerplate). If anything here ever
conflicts with the brand guide, the brand guide wins — update this file.

> Audience: parents of kids ages **6–16** in **Montgomery County, MD**.
> Parent register: reassuring, transparent, progress-oriented.

---

## Platform best practices (reviewed 2026)

These are the current rules for IG/FB Reels and feed posts. They change — when
you re-verify, update this section and the date in the heading.

**Hook (both platforms).** The first line has to land in the first 2–3 seconds —
a bold statement, a question, or a vivid image. Watch-time and completion rate
are what the algorithm ranks on, so the caption reinforces the video; it never
decorates it.

**Length.** Short and scannable for Reels (aim for the value in the first ~125
characters, before the "more" fold). Facebook tolerates longer storytelling than
IG, but for a Reel, tighter wins.

**Keywords beat hashtags.** Natural keywords woven into the caption ("youth
pickleball in Bethesda") now drive more discoverability than hashtag stuffing.
Write the keyword into the sentence first; add hashtags second.

**Hashtags — count is platform-specific:**
- **Facebook: 1–3.** Five or more is treated as a spam signal and *suppresses*
  reach. Never exceed 3 on FB.
- **Instagram: 3–5.** Light and relevant. The old "stuff 11–30 tags" advice is
  dead.
- **TikTok: 3–5.** Niche tags outperform broad ones; too many sends mixed
  signals to the algorithm.
- All three: prefer **local/community** tags (`#BethesdaMD`) over broad ones
  (`#sports`, `#love`). Skip any tag that wouldn't help a real person find or
  understand the video.

**Alignment.** Caption, hashtags, and cover frame should describe the same
thing — a cohesive signal to the algorithm about who should see it.

### TikTok is a search engine — treat the caption like SEO

TikTok ranks on keyword match across the caption, on-screen text, and the
spoken audio, not just hashtags. So on TikTok specifically:

- **Lead the caption with the literal topic keyword**, not a teaser or a
  question — e.g. "Youth pickleball evaluation in Montgomery County" before any
  flourish. Only the first ~100–150 characters show before the "more" fold, so
  the keyword has to be in sentence one.
- **Put the keyword on-screen in the first 2–3 seconds.** TikTok's OCR reads
  text overlays — a title card like "Free youth pickleball evaluation · MoCo"
  is a ranking signal the caption can't replace.
- **Say the keyword out loud in the first ~5 seconds** of the video. The
  speech layer is indexed too.
- Keep the caption short (150–300 chars is plenty); the video does the work.
- Hashtags: 3–5 niche/local, same banks below.

## Emoji rules (this is the #1 "looks like AI" tell)

- **1–3 per caption, maximum.** Sprinkled decorative emoji are the giveaway.
- **Functional only.** A `📍` before a location or a `🗓️` before a date earns
  its place. A 🔥/💪/🎾/⛳ dropped in for flavor does not. (And never use an
  emoji from the wrong sport — no golf flags, no tennis balls standing in for
  pickleball.)
- **Never substitute an emoji for a word** — screen readers can't parse it
  (accessibility), and it reads as gimmicky.
- **One closing accent max.** The tagline can carry a single 💚 if anything;
  usually cleaner with none.
- When in doubt, delete the emoji. Clean copy reads more premium and more human.

## The caption formula

1. **Hook** — one scroll-stopping line with the core keyword baked in
   (e.g. "Youth pickleball in Bethesda — your kid's next court is open").
2. **Proof** — what's actually happening in the clip (reps, cues, rallies,
   pathway, small groups). Show the work, don't hype it.
3. **Details (dated posts)** — location + date/time, one functional emoji each.
4. **One CTA** — see hierarchy below.
5. **Signoff** — the tagline: *Better than yesterday — together.*
6. **Hashtags** — 1–3 (FB) / 3–5 (IG), local-first.

EASE values (Ethics / Attitude / Skills / Excellence) get a nod only when they
land naturally — don't force all four.

## Voice rules (from `BRAND_GUIDELINES.md`)

**Words we use:** pathway, progress, reps, cues, EASE, triangle, community,
better than yesterday, evaluate, growth, family, next gen.

**Words/things we ban:**
- Hype without proof, sarcasm, insider slang, jargon.
- **No hard prices.** Never quote $25/monthly. The only live price is the $20
  drop-in, and it lives on `/schedule`, not in captions. Tease, don't quote.
- **Skill levels are Red / Orange / Green / Yellow** — never "Beginner / Pro."
- **Yellow Ball is invite-only** — never a public "register for Yellow Ball"
  CTA. Route tournament-track interest to `/yellowball/inquiry`.
- Ages **6–16 is strict** — no under-6, no "we'll make an exception."

**CTA hierarchy — one primary CTA per post:**
- **Cold / awareness post →** Get a Free Evaluation → "link in bio"
  (`/free-evaluation`).
- **Dated session-fill post →** Save your spot → "link in bio" (`/schedule`),
  with a free evaluation as the soft inline fallback for first-timers.
- Set the actual bio link to match whichever CTA the post uses.
